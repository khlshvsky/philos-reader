# Философский канон

Трекер чтения 120 главных книг по философии — от Платона до наших дней.  
Один час в день, хронологический порядок.

## Демо

Задеплоить на Vercel одной кнопкой:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/ВАШ_ЮЗЕРНЕЙМ/philos-reader)

## Структура

```
philos-reader/
├── index.html    — разметка страницы
├── style.css     — стили (EB Garamond + Inter, тёмная тема)
├── books.js      — 120 книг с метаданными
├── app.js        — логика: поиск, фильтры, прогресс
└── vercel.json   — настройки деплоя
```

## Возможности

- 120 книг в 7 эпохах: Античность, Отцы церкви, Средневековье, Возрождение, Новое время, XIX век, XX–XXI
- Поиск по названию и автору
- Фильтрация по эпохе
- Отметка прочитанных книг (сохраняется в `localStorage`)
- Счётчик часов и прогресс-бар
- Адаптивный дизайн, тёмная тема

## Деплой на Vercel

```bash
npm i -g vercel
vercel
```

Или через [vercel.com](https://vercel.com) — перетащи папку проекта, готово.

## Деплой на GitHub Pages

1. Создай репозиторий на GitHub
2. Запушь файлы
3. Settings → Pages → Source: `main` / `root`
